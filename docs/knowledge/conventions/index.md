# Conventions

Repository-wide conventions. Do not restate configuration values that are already in `pyproject.toml` or other config files — record the **judgment criteria and exception handling** that those values cannot express.

`AGENTS.md` carries the short summary that humans and AI agents skim first. Documents in this directory expand the *why* behind those rules and clarify how to handle edge cases.
